Code

def generate_toc(soup):
    h3s = []
    for i, h3 in enumerate(soup.find_all('h3')):
        h3['id'] = 'sec-{0}'.format(i)
        h3s.append(h3.get_text())
    if h3s == []: return ""

    out = "<div class='toc'><h3>Table of Contents</h3><ul>\n"
    for i, h3 in enumerate(h3s):
        out += "<li><a href='#sec-{0}'>{1}</a></li>\n".format(i, h3)
    out += "</ul></div>\n"
    return out

Using

Cmd arguments

I also added some command-line arguments

$ python bloggen.py -h
usage: bloggen.py [-h] [-g GENERATE] [-n] [-s] [-q]

optional arguments:
  -h, --help            show this help message and exit
  -g GENERATE, --generate GENERATE
  -n, --no-index        don't generate index
  -s, --skip-files      skip files beginning with . # or -
  -q, --quiet

So it's now possible to regenerate post files individually, perhaps combined with a tool like make.

I may add this functionality to bloggen.py itself.
