bloggen.py source and instructions


2015-09-17 by dvolk

I've cleaned up the blog generator source a bit and uploaded it to github.

Github repo.

bloggen.py is configured by editing bloggen.py 😸.

Posts come from ./content/posts/*.

The generated site is in ./generated/*.

Posts are formatted using HTML (initially I used markdown, but I think HTML is more convenient).

For example:

<span class="title">title here</span>
<span class="time">17/9/3000 16:00</span>
<span class="author">you!</span>
<div class="summary">
  <p>This is the summary that apperas both on the
    index and the post page.</p>
</div>
<div class="rest">
  <p>This is the rest of the post. It only appears
    on the post page</p>
</div>

There are 3 special spans, "title", "time" and "author" and two special divs "summary" and "rest". The time must be in a dd/mm/yyyy hh:mm format by default. The content of the "summary" div is displayed both on the index and the seperate post page. The content of the "rest" div is only displayed on the post page.

None of these elements are required. A minimum post is just an empty file. Anything outside the elements is ignored.

That's about it!
